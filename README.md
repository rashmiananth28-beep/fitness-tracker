# 💜 Rashmi's Fitness Tracker

A personal mobile-first fitness tracker built as a Progressive Web App. Designed for a **petite intermediate female** (148cm) with goals of toning, lean muscle, and strength.

## 🌐 Live App
**[Open on your phone → rashmiananth28-beep.github.io/fitness-tracker](https://rashmiananth28-beep.github.io/fitness-tracker)**

> Bookmark this on your phone's home screen for instant access!

## ✨ Features
- 📅 6-day workout plan tailored for petite frame — heavy compound lifts + isolation
- ✅ Tick off exercises as you go (warm-up → main → cool-down)
- 💧 Water intake tracker with +250ml / +500ml / +1L buttons
- 👟 10,000 step goal tracker
- 🔥 Workout streak counter
- 📝 Daily log — record duration, type, notes and PRs
- 🌙 Sunday rest day
- 💡 Daily coaching tip for each session
- 📱 Fully mobile-first — works like an app on your phone

## 📱 Add to Home Screen (use like an app!)
**iPhone:** Open in Safari → Share → Add to Home Screen  
**Android:** Open in Chrome → Menu (⋮) → Add to Home Screen

## 🛠 Tech Stack
HTML5, CSS3, Vanilla JavaScript, localStorage for data persistence
